# lps_nextodonto

Landing page estatica da NextOdonto.

## Como rodar

```bash
npm start
```

Acesse `http://localhost:3000`.

## Validacao

```bash
npm run validate
```

O validador confere se o `index.html` tem doctype, se as ancoras internas apontam para IDs existentes, se os assets referenciados existem e se o JavaScript inline nao tem erro de sintaxe.
